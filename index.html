Music Genre Influence Networks

Mapping Artist Collaboration & Linguistic Diffusion across the music landscape

What is the idea?

We build a network of artists connected by collaborations, shared features, or co-listening patterns to study genre evolution and cross-genre influence. This helps uncover how musical styles evolve, how artists influence each other, and how language flows through genres.

Data Sources

  • Zenodo Dataset (2021): Global artist collaboration networks (2017–2019)
  • Spotify-derived metadata: Artist popularity, genres, followers, and other attributes (from precompiled source)
  • Lyrics data: Integrated in the artist dataset, not extracted manually

This dataset was chosen because API access (Spotify, Genius) was highly restricted or impractical for large-scale collection. Alternative dumps required too much time and manual cleaning. After spending ~3 days experimenting, we decided to use a reliable, pre-cleaned dataset from Zenodo that supports our analysis goals.

Network & Text Integration

Artists are represented as nodes. Edges represent collaborations or similarity in listeners. Lyrics are analyzed using NLP techniques to extract sentiment, topic patterns, and linguistic uniqueness. These are then overlaid onto the network.

Dataset Summary

  • Total size: 3 yearly networks (2017–2019), 680–1067 links each
  • Attributes: Genre, popularity, lyrics, number of songs
  • Lyrics volume: 50–200 songs per artist
  • Artist info table: 3,612 artists with metadata

Preliminary Insights

  • Genre clusters are identifiable via community detection
  • High-degree artists act as bridges across genres
  • Lyrical traits appear to diffuse along collaboration edges
