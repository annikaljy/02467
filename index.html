Music Genre Influence Networks

Mapping Artist Collaboration & Linguistic Diffusion across the music landscape

What is the idea?

We build a network of artists connected by collaborations, shared features, or co-listening patterns to study genre evolution and cross-genre influence. This helps uncover how musical styles evolve, how artists influence each other, and how language flows through genres.

Data Sources

  • Spotify API: Artist metadata, genres, popularity, collaborations
  • Genius API: Song lyrics

Data was collected using Python tools that interact with these APIs.

Network & Text Integration

Artists are represented as nodes. Edges represent collaborations or similarity in listeners. Lyrics are analyzed using NLP techniques to extract sentiment, topic patterns, and linguistic uniqueness. These are then overlaid onto the network.
